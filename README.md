# AI Job Market Trends Analysis 2026

## Overview

This project analyzes over 10,000 AI and Machine Learning job postings to uncover hiring trends, salary distributions, skill requirements, and workforce demands across different countries and industries.

Using Exploratory Data Analysis (EDA), the project provides insights into the rapidly growing AI job market and helps identify the most valuable skills, job roles, and career opportunities.

---

## Problem Statement

The AI industry is evolving rapidly, creating significant demand for professionals with Machine Learning, Deep Learning, Cloud Computing, Python, and Data Science skills.

This project aims to answer important questions such as:

* Which AI roles are most in demand?
* Which countries offer the highest AI opportunities?
* What skills are employers seeking?
* How does salary vary across experience levels?
* Are remote jobs becoming more common?
* Which industries hire the most AI professionals?
